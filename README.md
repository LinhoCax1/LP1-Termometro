# Avaliação Zero - Lista Termômetro

Este repositório contém os programas desenvolvidos para a **Avaliação Zero - Lista Termômetro**. Todos os programas foram escritos em **C++** e compilados utilizando o compilador `g++`.

## Requisitos
Para compilar e executar os programas é necessário ter instalado:
- Compilador `g++`
- Terminal ou prompt de comando

## Como compilar e executar

Para compilar um dos arquivos `.cpp`, execute no terminal:

`g++ nome-do-arquivo.cpp`

Isso irá compilar o código e gerar um arquivo executável chamado:

`a.exe`

Após compilar o programa, execute o seguinte comando:

`./a.exe`

Esse comando executa o programa que foi gerado pela compilação.

## Estrutura

Os arquivos `.cpp` presentes no repositório correspondem às soluções dos exercícios da lista. Cada programa pode ser compilado e executado individualmente seguindo os passos descritos acima.
